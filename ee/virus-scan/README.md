# virus-scan

扫描上传文件，判断是否有安全隐患

## 如何使用

1、`.evn.local` 配置环境变量 `VIRUS_SCAN_SCAN_COMMAND=clamdscan`

2、系统安装此软件包

环境变量配置扫描病毒的命令，如未配置则认为无需扫描
